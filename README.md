# Titan Personal Assistant

Titan is a voice-controlled personal assistant with face recognition capabilities. It can help you with various tasks including system control, web searches, and more.

## Features

1. **Face Recognition** - Recognizes registered users and sends alerts for unknown faces
2. **Voice Commands** - Responds to various voice commands
3. **System Control** - Volume, brightness, battery status, WiFi, shutdown/reboot
4. **Web Integration** - Google search, YouTube, Spotify
5. **Application Control** - Open/close applications
6. **Weather Information** - Get current weather for your location
7. **Music Playback** - Play music from your library
8. **User Management** - Add/delete user faces, list registered users
9. **MQTT Integration** - Control smoke detector buzzer

## Available Commands

- "What can you do?" or "Help" - List all capabilities
- "Who are you?" - Introduction
- "Hello" - Greeting
- "Time" - Get current time
- "Volume" commands - Control system volume
- "Brightness" commands - Control screen brightness
- "Battery percentage" - Check battery status
- "Search [query]" - Search on Google
- "Open YouTube/Spotify" - Search on YouTube or Spotify
- "Open/Close application" - Open or close apps
- "Weather" - Get weather information
- "Play music" - Play music from your library
- "Add new face" - Register a new user
- "Delete face" - Remove a user (requires passcode)
- "List users" - Show registered users
- "Where is my smoke detector?" - Ring the buzzer
- "Shutdown/Reboot" - System power control
- "Stop" - Exit the assistant
